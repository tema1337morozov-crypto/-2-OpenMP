# Лабораторная работа №2: Параллельное умножение матриц с OpenMP

## Описание

Программа умножает две квадратные матрицы, используя технологию OpenMP для параллельных вычислений. Проводится сравнение производительности при разном количестве потоков (1, 2, 4, 8).

## Требования

- Компилятор C++ с поддержкой OpenMP (MinGW-w64, g++)

## Сборка и запуск

- `g++ main_omp.cpp -o omp_lab.exe -fopenmp -O2`
- `./omp_lab.exe`

## Генерация больших матриц 

- `python generate_matrix.py 400`

## Если есть проблема с символами при запуске, введите сначала эту команду
- `chcp 65001`
